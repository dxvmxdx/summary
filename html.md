# 1. block과 inline

HTML에는 두가지 종류의 요소가 있다.

<br>

## 1-1. block-level element

- 사용 가능한 전체 너비를 갖는다.
- `width`와 `height` 속성 적용 가능
- `margin`과 `padding` 속성 적용 가능

<br>
<br>
<br>

## 1-2. inline-level element

- conten 만큼의 너비를 갖는다.
- `width`와 `height` 속성이 적용되지 않는다.
- `margin`과 `padding`의 상하 속성이 적용되지 않는다.  
  (엄밀히 말하면 `padding`의 상하 속성은 적용되나, 레이아웃 공간에 영향을 주진 않는다.)

<br>

### inline-block element

- content 만큼의 너비를 갖는다.
- `width`와 `height` 속성 적용 가능
- `margin`과 `padding` 속성 적용 가능
