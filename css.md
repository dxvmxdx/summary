# 1. box model

### border와 outline 차이

- border  
  요소 테두리
- outline  
  요소 외곽선 (`border` 바깥에 그려지는 선)

<br>

### box-sizing

요소의 너비와 높이를 계산하는 방법을 지정

- `content-box`  
  기본값. 요소의 너비와 높이가 `content` 자체
- `border-box`  
  요소의 너비와 높이를 `border`까지 포함
